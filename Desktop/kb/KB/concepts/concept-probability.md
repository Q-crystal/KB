# Intuition
概率是对事件发生可能性的定量描述，取值范围在0到1之间。例如，抛硬币正面朝上的概率是0.5。

# Rigour
- **定义**: 概率是样本空间中事件发生的可能性度量
- **假设**: 样本空间中的每个结果是等可能的
- **适用范围**: 随机事件、不确定性分析、统计推断
- **代价或复杂度**: 计算复杂事件的概率可能需要组合数学或积分
- **失败模式**: 样本空间定义错误、事件独立性假设不成立

# Evidence
概率的概念在概率论和统计学中被广泛接受，是许多学科的基础。
