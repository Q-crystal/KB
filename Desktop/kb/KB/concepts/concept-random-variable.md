# Intuition
随机变量是将随机试验的结果映射到数值的函数。例如，抛硬币的结果可以映射为0（反面）和1（正面）。

# Rigour
- **定义**: 随机变量是从样本空间到实数集的可测函数
- **假设**: 基础概率空间已定义
- **适用范围**: 概率分布、统计模型、风险管理
- **代价或复杂度**: 处理连续型随机变量需要微积分
- **失败模式**: 分布假设错误、变量独立性假设不成立

# Evidence
随机变量的概念是概率论的核心，被广泛应用于各个领域。
