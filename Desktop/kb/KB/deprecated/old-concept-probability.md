# Intuition
旧概率概念是对事件发生可能性的描述，取值范围在0到1之间。

# Rigour
- **定义**: 概率是事件发生的可能性
- **假设**: 事件是等可能的
- **适用范围**: 简单随机事件
- **代价或复杂度**: 计算简单
- **失败模式**: 无法处理复杂事件
