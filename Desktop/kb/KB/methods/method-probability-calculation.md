# Intuition
概率计算方法是用于计算事件发生概率的技术，包括古典概率、几何概率、条件概率等。例如，计算掷骰子点数大于4的概率是2/6=1/3。

# Rigour
- **定义**: 概率计算方法是基于概率公理和规则的计算技术
- **假设**: 样本空间已定义，事件明确
- **适用范围**: 概率问题求解、风险评估、决策分析
- **代价或复杂度**: 复杂问题可能需要组合数学或积分
- **失败模式**: 样本空间定义错误、事件独立性假设不成立

# Evidence
概率计算方法在概率论和统计学教材中被详细介绍，是解决概率问题的基础。
